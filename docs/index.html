FCA Store v0

Versioned formal concept analysis store with lattice build and runtime query APIs.

Core Python API

  • put_incidence(version_id, objects, attributes, edges_present, edges_absent=None, meta=...)
  • build_lattice(version_id, method="nextclosure|ganter")
  • get_extent(version_id, attribute_ids)
  • get_intent(version_id, object_ids)
  • suggest_split_attributes(version_id, current_attribute_ids, remaining_object_ids=None)
  • diff_versions(v1, v2)

HTTP Endpoints

  • GET /versions
  • POST /versions/{id}/extent
  • POST /versions/{id}/intent
  • GET /versions/{id}/lattice/status
  • GET /versions/{id}/diff/{other}

Storage Choice (v0)

SQLite is used for v0 simplicity and deterministic local execution. The interface is backend-agnostic so PostgreSQL can be added later without changing callers.
